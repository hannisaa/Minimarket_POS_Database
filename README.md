# Minimarket Point of Sales Database

## Overview

This project is a relational database system designed for a Minimarket Point of Sales (POS) application.

The database manages:

- Product inventory
- Supplier information
- Employee data
- Sales transactions
- Transaction details
- Reporting system

## Technologies

- MySQL
- XAMPP
- phpMyAdmin

## Database Features

### Tables

1. Category
2. Employee
3. Product
4. Supplier
5. Transaction
6. Transaction Detail

### Views

- Sales Report
- Sales Per Period
- Low Stock Products
- Transaction Detail
- Employee Performance

### Stored Procedures

- Add Employee
- Add New Product
- Add To Cart
- New Transaction
- Update Stock

### Triggers

- Check Stock
- Prevent Duplicate Product Name
- Prevent Transaction History Deletion
- Employee Validation
- Transaction Validation
